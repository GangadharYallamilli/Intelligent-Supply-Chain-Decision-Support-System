# 🚚 Intelligent Supply Chain Decision Support System

## 📌 Project Description

An end-to-end Supply Chain Analytics and Decision Support System developed using Python, Machine Learning, MySQL, and Microsoft Power BI to improve operational efficiency, demand forecasting, delay risk analysis, and business decision-making.

This project combines Data Analytics, Predictive Modeling, Simulation, SQL Integration, and Business Intelligence into a single intelligent platform for modern supply chain management.

---

## 📖 Project Overview

The system helps organizations to:

- ✅ Forecast product demand
- ✅ Predict shipment delay risks
- ✅ Analyze operational performance
- ✅ Simulate supply chain disruptions
- ✅ Generate intelligent business recommendations
- ✅ Visualize KPIs through interactive dashboards

The project demonstrates how Machine Learning and Business Intelligence can support real-world supply chain optimization and strategic decision-making.

---

## ⚙️ Core Modules

### 🧹 Data Cleaning & Preprocessing
- Handled missing values and duplicate records
- Data transformation and normalization
- Encoded categorical variables
- Prepared datasets for machine learning

### 🏗️ Feature Engineering
- Delivery performance indicators
- Delay categories
- Inventory metrics
- Supplier performance analysis
- Demand-related attributes

### 📊 Exploratory Data Analysis (EDA)
- Demand trends
- Delay distributions
- Shipping patterns
- Supplier performance
- Regional sales analysis

### 📈 Demand Prediction
Implemented Multiple Linear Regression for forecasting future product demand.

### 🌲 Delay Risk Prediction
Built a Random Forest Classifier to predict shipment delay risks.

### 🔄 Supply Chain Simulation
Simulated:
- Supplier disruptions
- Transportation delays
- Demand spikes
- Inventory shortages

### 🧠 Decision Recommendation Engine
Recommends actions like:
- Reordering stock
- Changing suppliers
- Increasing inventory buffers
- Optimizing delivery schedules

### 🗄️ SQL Database Integration
Integrated with MySQL for:
- Data storage
- Query execution
- Reporting support

### 📉 Power BI Dashboard
Interactive dashboards for:
- KPI monitoring
- Demand forecasting
- Delay risk analysis
- Supplier tracking
- Business reporting
